2021 gdut 图形学与人机交互技术 课程设计

IDE: Visual Studio 2017           language:C++，opengl

依赖库：assimp ，glm ，stb_image ，glfw , glew

老师在课上一般只会对书本讲课，讲的也是旧的OpenGL核心渲染，教材也是旧OpenGL，然后最后让你做一个课设，感觉最多只能学一些图形学知识和算法。（如果是真心想学OpenGL，那么课堂内容反而没用，还得自己去学比较现代的OpenGL，例如着色器之类的）
无论是旧的还是现代的opengl，网上的中文资料都很少。而且由于opengl较为灵活，和我们认知的调用API可能有所不同，属于是画一个三角形都要近100行的代码的离谱。
此外，现在中国的有关图形学的就业比较窄，就业少，不怎么符合本科生的就业。
大多数图形学就业都需要 扎实的图形学知识 + 一定的编程基础和图形学算法 + 主流的图形学工具（OpenGL不是，OpenGL往往只是岗位加分项） + 项目背景知识
综上所述，选这门课程大多都是浪费时间，也不明白为什么会开这个课程，作者十分不推荐选这门课。
